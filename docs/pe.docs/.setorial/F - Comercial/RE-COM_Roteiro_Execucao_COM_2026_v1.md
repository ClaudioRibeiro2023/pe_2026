## 1. Contexto e princípios do Roteiro

O RE-COM é a Camada 3 do PE2026 para o Comercial — o roteiro operacional que traduz as iniciativas do DOC 11-F.2 em ações concretas para o WBR semanal. Cada iniciativa tem responsável com nome, custo estimado, semáforo de status e critério de encerramento verificável.

> **Camada 3 é aditiva.** O RE-COM não altera os documentos da Camada 1 (estratégia) nem da Camada 2 (programas). Apenas operacionaliza.

| Princípio | Aplicação no RE-COM |
|---|---|
| Dono com nome | Cada INIT tem 1 pessoa nomeada — não "área" |
| Custo estimado | Faixa de investimento por INIT — dentro do envelope R$ 770K–1.000K |
| Semáforo | Verde · Amarelo · Vermelho — atualizado todo WBR |
| Evidência verificável | O que comprova conclusão — não narrativa de progresso |
| Prazo explícito | Data de início e marco de fim — sem "em andamento" indefinido |
| Faseamento | COM-P1 é pré-condição. COM-P2/P3/P4 ativam com liderança contratada |
| Linguagem operacional | "Limite de gasto" em vez de "envelope"; "quem faz" em vez de "RACI" |

## 9. Glossário operacional

| Termo operacional | Equivalente PE | O que significa na prática |
|---|---|---|
| Quem faz | RACI / Sponsor-Dono | Nome da pessoa responsável — não "área" |
| Limite de gasto | Envelope orçamentário | Quanto pode gastar no máximo nesta INIT |
| Prioridade P0/P1/P2 | Tier de priorização | P0: inegociável · P1: alto impacto · P2: desejável se couber |
| Semáforo | Status de execução | Verde: ok · Amarelo: risco · Vermelho: bloqueado |
| Prova de conclusão | Evidência (EVID-*) | Documento, log, relatório que comprova que foi feito |
| Critério de pronto | Critério de encerramento | Condição objetiva para considerar a INIT concluída |
| Funil | Pipeline com etapas | Prospecção → qualificação → proposta → negociação → fechamento |
| CRM | Sistema comercial | Ferramenta que registra 100% das interações com prospects/clientes |
| Forecast | Previsão de receita | Estimativa de receita por horizonte 30/60/90 dias |
| Gatilho Q3 | Decisão condicional | Se pipeline Aero < 4 em junho → Manutenção; ≥ 4 → Expansão |
| Handoff | Transferência | Passagem formal de contato/processo entre CS e Comercial |
| WBR | Revisão semanal | Reunião rápida de status — pipeline, ações, bloqueios |
| MBR | Revisão mensal | Reunião de KPIs, decisões e ajustes de curso |
| QBR | Revisão trimestral | Reunião estratégica — objetivos, cenários, estrutura |
