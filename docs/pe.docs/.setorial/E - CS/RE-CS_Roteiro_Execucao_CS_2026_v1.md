## 1. Contexto e princípios do Roteiro

O RE-CS é a Camada 3 do PE2026 — o roteiro operacional que traduz as iniciativas do DOC 11-E.2 em ações concretas para o WBR semanal (Sala de Situação). Cada iniciativa tem responsável com nome, custo estimado, semáforo de status e critério de encerramento verificável.

> **Camada 3 é aditiva.** O RE-CS não altera os documentos da Camada 1 (estratégia) nem da Camada 2 (cadernos setoriais). Apenas operacionaliza. Ajustes estruturais vão para o DOC 11-E.1 ou 11-E.2 — não aqui.

| Princípio | Aplicação no RE-CS |
|---|---|
| Dono com nome | Cada INIT tem 1 pessoa nomeada — não "área" |
| Custo estimado | Faixa de investimento por INIT — dentro do envelope R$ 305–465K |
| Semáforo | Verde · Amarelo · Vermelho — atualizado todo WBR |
| Evidência verificável | O que comprova conclusão — não narrativa de progresso |
| Prazo explícito | Data de início e marco de fim — sem "em andamento" indefinido |
| Linguagem operacional | "Limite de gasto" em vez de "envelope orçamentário"; "quem faz o quê" em vez de "RACI" |

| Composição envelope total CS 2026 | Faixa (R$) |
|---|---|
| Folha CS (2-3 pessoas parcialmente dedicadas) | R$ 180–250K |
| Iniciativas do roteiro (sem folha) | R$ 50–95K |

| Viagens de relacionamento | R$ 30–50K |

> **Safety floor:** se margem cair abaixo de 20%, iniciativas P2 pausadas imediatamente. P1 revisadas. P0 continuam (Sala de Situação e ativação não param).

## 9. Glossário operacional

|---|---|---|
| Limite de gasto da área | Envelope orçamentário | Valor máximo de despesa aprovado para CS no ano |

| Critério de encerramento | Definition of Done (DoD) | O que precisa existir para considerar a INIT concluída |
| Prova de execução | Evidência (EVID) | Documento, painel ou registro verificável de que a entrega foi feita |
| Semáforo | Status visual WBR | Verde/Amarelo/Vermelho — estado atual da iniciativa |
| Dono | Accountable | 1 pessoa responsável pelo resultado e pela evidência |
| Sponsor | Patrocinador | Quem garante prioridade e destrava bloqueios — nível Direção |
| War Room | Sala de Situação | Rito semanal de gestão de saldo, vazão e prioridades Pareto |
| WBR | Revisão semanal | O que foi feito, o que está bloqueado, próximas ações |
| MBR | Revisão mensal | Placar de KPIs, metas, decisões e backlog de ações |

| INIT | Iniciativa | Ação concreta com dono, prazo, custo e evidência definidos |
| P0 / P1 / P2 | Prioridade | P0: crítico 90 dias · P1: estratégico do ano · P2: melhoria |

| Health score | Saúde do contratante | Índice composto que sinaliza risco de churn |
