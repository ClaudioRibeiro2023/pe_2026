## 1. Contexto e princípios do Roteiro

O RE-RH é a Camada 3 do PE2026 — o roteiro operacional que traduz as iniciativas do DOC 11-A.2 em ações concretas para o WBR semanal. Cada iniciativa tem responsável com nome, custo estimado, semáforo de status e critério de encerramento verificável.

> **Camada 3 é aditiva.** O RE-RH não altera os documentos da Camada 1 (estratégia) nem da Camada 2 (programas). Apenas operacionaliza. Ajustes estruturais vão para o DOC 11-A.1 ou 11-A.2 — não aqui.

| Princípio | Aplicação no RE-RH |
|---|---|
| Dono com nome | Cada INIT tem 1 pessoa nomeada — não "área" |
| Custo estimado | Faixa de investimento por INIT — dentro do envelope R$ 600–800K |
| Semáforo | Verde · Amarelo · Vermelho — atualizado todo WBR |
| Evidência verificável | O que comprova conclusão — não narrativa de progresso |
| Prazo explícito | Data de início e marco de fim — sem "em andamento" indefinido |
| Linguagem operacional | "Limite de gasto" em vez de "envelope orçamentário"; "quem faz o quê" em vez de "RACI" |
