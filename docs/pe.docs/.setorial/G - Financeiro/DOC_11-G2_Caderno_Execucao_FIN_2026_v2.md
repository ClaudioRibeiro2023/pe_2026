## 2. Propósito deste Caderno

O Caderno Execução (11-G.2) detalha as iniciativas que materializam os OKRs/KRs definidos no Caderno Direção (11-G.1). Cada iniciativa é apresentada em formato de **Cartão Didático com 14 campos** padronizados, garantindo clareza operacional e rastreabilidade.

Este caderno é a referência para o WBR (acompanhamento semanal) e alimenta diretamente o Roteiro de Execução (RE-FIN v1).
