## 1. Contexto e princípios do Roteiro

O RE-MKT é a Camada 3 do PE2026 — o roteiro operacional que traduz as iniciativas do DOC 11-B.2 em ações concretas para o WBR semanal. Cada iniciativa tem responsável com nome, custo estimado, semáforo de status e critério de encerramento verificável.

> **Camada 3 é aditiva.** O RE-MKT não altera os documentos da Camada 1 (estratégia) nem da Camada 2 (programas). Apenas operacionaliza. Ajustes estruturais vão para o DOC 11-B.1 ou 11-B.2 — não aqui.

| Princípio | Aplicação no RE-MKT |
|---|---|
| Dono com nome | Cada INIT tem 1 pessoa nomeada — não "área" |
| Custo estimado | Faixa de investimento por INIT — dentro do envelope MKT |
| Semáforo | Verde · Amarelo · Vermelho — atualizado todo WBR |
| Evidência verificável | O que comprova conclusão — não narrativa de progresso |
| Prazo explícito | Data de início e marco de fim — sem "em andamento" indefinido |
| Linguagem operacional | "Limite de gasto" em vez de "envelope"; "quem faz o quê" em vez de "RACI" |

## 9. Glossário operacional

| Termo operacional | Equivalente PE2026 | Definição |
|---|---|---|
| Limite de gasto da área | Envelope orçamentário | Valor máximo de despesa aprovado para o programa no ano |
| Quem faz o quê | RACI | Tabela de papéis: quem executa, quem decide, quem apoia |
| Critério de encerramento | Definition of Done (DoD) | O que precisa existir para considerar a INIT concluída |
| Prova de execução | Evidência (EVID) | Documento, painel ou registro verificável de que a entrega foi feita |
| Semáforo | Status visual WBR | Verde/Amarelo/Vermelho — estado atual da iniciativa |
| Dono | Accountable | 1 pessoa responsável pelo resultado e pela evidência |
| Sponsor | Patrocinador | Quem garante prioridade e destrava bloqueios — nível Direção |
| WBR | Revisão semanal | O que foi feito, o que está bloqueado, próximas ações |
| MBR | Revisão mensal | Placar de KPIs, metas, decisões e backlog de ações |
| Roteiro (RE) | Camada 3 | Documento operacional que detalha cada iniciativa para o WBR |
| INIT | Iniciativa | Ação concreta com dono, prazo, custo e evidência definidos |
| P0 / P1 / P2 | Prioridade | P0: crítico 90 dias · P1: estratégico do ano · P2: melhoria |
| CPL | Custo por Lead | Investimento de Marketing dividido pelo número de leads qualificados |
| ROI | Retorno sobre investimento | Resultado financeiro por ação — obrigatório em feiras |
| Pipeline | Funil de oportunidades | Conjunto de oportunidades em diferentes etapas de conversão |
