## 8  Glossário Operacional

| Termo operacional | Equivalente PE | Definição simples |
|-------------------|---------------|-------------------|
| Plano da semana | Planejamento de capacidade | Documento publicado na sexta com equipes e volumes da semana seguinte |
| Meta de hectares | KR (Key Result) | Número de hectares a executar no período |
| Checklist de campo | SLA | Lista de verificação para qualidade |
| Retrabalho | Não-conformidade | Hectares re-executados por problema |
| Sala de Situação | War room | Reunião semanal de priorização |
| Semáforo | Status INIT | Verde/Amarelo/Vermelho/Branco |
| Top-14 | Pareto Top-14 | 14 contratantes com maior saldo |
| Ponto de passagem | Handoff | Entrega formal entre áreas |
| Run-rate | Vazão | Hectares por semana |
| Dashboard | Painel de gestão | Tela com indicadores atualizados |
| COO | Diretor de Operações | Liderança operacional |
| Envelope | Limite de gasto | Valor máximo da área no ano |
