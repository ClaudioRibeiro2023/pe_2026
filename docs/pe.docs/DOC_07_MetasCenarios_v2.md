## Conclusão

O exercício 2026 parte de posição de força: margem de 34,3%, base contratual cobrindo 79% da meta e estrutura comprovadamente eficiente. O cenário base projeta crescimento de 10,8% com margem no guardrail de 30%, absorvendo investimentos estratégicos sem comprometer saúde financeira. O pessimista, mesmo com perda regulatória, é gerenciável com contingência de R$ 828K. Os gatilhos estão calibrados para mover entre cenários com método. O Q3 concentra as decisões mais importantes: Aero (expandir ou manter) e regulatório (confirmação ou contingência).

## Glossário

| Termo | Significado |
|-------|-----------|
| Cenário base | Projeção 60% probabilidade, premissas médias |
| Cenário pessimista | Pior caso razoável (15%), gatilho regulatório |
| Cenário otimista | Melhor caso razoável (25%), expansão + upsell |
| Valor esperado | Média ponderada por probabilidade |
| Receita travada | Receita contratada (renovações + plurianuais) |
| GAO | Grau de alavancagem operacional |
| Safety floor | Margem mínima (20%) antes de contingência |
| Guardrail | Margem-alvo mínima (30%) para investimento |
| HHI | Índice de concentração de receita |
| Early Warning | Indicadores antecipados de risco |
| Contingência | Plano de redução para cenário adverso |
| PMR | Prazo médio de recebimento |
