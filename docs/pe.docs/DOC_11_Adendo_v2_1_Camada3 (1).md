# DOC 11 — Adendo v2.1
## Seção 13: Roteiros de Execução (Camada 3)

Versão: **2.1** | Data: **02/mar/2026**

> Seção nova — define a terceira camada do PE e o papel da Assessoria.

---

### 13.1  O que são
Roteiros de Execução (RE) traduzem os cadernos setoriais em passos práticos com linguagem acessível, nomes de responsáveis e cronograma real. Cada área = 1 Roteiro: RE-RH, RE-MKT, RE-P&D, RE-OPS, RE-CS, RE-COM, RE-FIN.

### 13.2  Caderno setorial × Roteiro de Execução
| | Caderno (C2) | Roteiro (C3) |
|-|-------------|-------------|
| Serve para | Governança | Execução dia a dia |
| Quem lê | Direção + Líder | Líder + Time |
| Linguagem | Institucional, códigos | Prática, nomes |
| Quem produz | Direção (PE) | Assessoria + Líder |
| Muda metas? | Sim (decisão formal) | Não — herda |

### 13.3  10 campos obrigatórios
Código, resultado vinculado, programa, meta, custo, indicador, trimestre, responsáveis (nomes), evidência, status.

### 13.4  Assessoria
Traduz, constrói com líder, facilita ritos, mantém atualizado, alimenta Placar. Não cria metas, não muda códigos, não assina, não substitui líder.

### 13.5  Governança
WBR (semanal) → MBR (mensal) → QBR (trimestral). Roteiro alimenta MBR. MBR alimenta Placar 11-H.
